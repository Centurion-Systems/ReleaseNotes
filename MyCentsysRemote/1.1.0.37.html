Release Notes: MyCentsys Remote

Release version: 1.1.0.37

Release date: July 20, 2023


Details

  • Added the ability to view operator admins on D Smart and SDO4 Smart operators
  • MyCentsys Remote will ensure that Smart operators time is set correctly during activations.
  • Added the option for users to back up their app settings, customisations, favourites and added operators to the cloud, so that is can be restored when needed.
  • Added a request to ignore battery optimisation, to ensure that notifications are delivered in Realtime. (Android only)
