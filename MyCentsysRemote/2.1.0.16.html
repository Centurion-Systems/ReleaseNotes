Release version: 2.1.0.16

Release date: February 03, 2025

Details

Features

  • SMART Device Notifications with Text-to-Speech: Users can now receive spoken alerts for SMART device notifications, enhancing accessibility and convenience.
  • Redesigned Action/Status Icons for ULTRA Devices: A fresh design provides clearer visual feedback on device actions and statuses.
  • High-Contrast Mode for SMART Icons: Improves visibility and accessibility.
  • Search Functionality in App Settings & Action Status Selection: Users can now quickly locate settings and actions within the app.
  • Filter by Device Type in Home Tab: Users can now filter their device list based on type, simplifying navigation.
  • Text-to-Speech Toggle in App Settings: Allows users to enable or disable text-to-speech functionality based on their preference.

Improvements

  • Resolved Network Balance Issue: Fixed an issue where network balances were not displaying if the device was not using a Centurion SIM.
  • Improved BLE Trigger Speed: Optimised BLE triggers for faster activation.
  • Multiple UI, Performance, and Stability Enhancements: Various refinements to improve the overall user experience.
