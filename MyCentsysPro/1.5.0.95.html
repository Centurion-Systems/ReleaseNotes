Release version: 1.5.0.95

Release date: December 8, 2023

Details

App

  • Data Usage Declaration:
  • Users can now review the data usage and reasons before entering any details, following OTP verification.
  • Improved stability for enhanced performance.

DX Features

  • Origin Philosophy Change
  • Origin used for gate calibration.
  • In lost state, the gate moves towards the reference end-stop on the first trigger.
  • Operators exit lost state by moving the full gate length in any direction, with or without an origin.
  • If a magnet goes missing, all operators run at half-speed, using end-stops instead of the origin.

Known Issue

  • The gate does not retract after stopping on a closing beam break while in the lost state.

VX Features

  • Trigger Schedules:
  • Users can now set schedules for triggering specific actions, enhancing automation and control based on predefined time frames.
  • In lost state, the gate moves towards the reference end-stop on the first trigger.
  • ChronoGuard:
  • Introduction of ChronoGuard, allowing users to set remotes or SMARTGUARDair channels to function at specific times.
  • Stop on Open for PIRAC:
  • A new feature that enables the system to stop on opening for PIRAC, enhancing security or controlling gate movements.
  • PIRAC for FRX:
  • PIRAC functionality extended to FRX, enabling users to activate PIRAC on FRX as available on other triggers.
  • FRX Trigger on Home Screen:
  • Users can now trigger FRX directly from the home screen, providing quick and convenient access to test this feature.
