Release version: 1.5.0.123

Release date: January 23, 2025

Details

Features

  • Enhanced Limit Setup Screens: Improved the limit setup screens to warn if the controller detects that an Origin Sensor is plugged in but the magnet on the gate is not detected.
  • Third-Party Solar Charger Support: Added support for the CENTURION (third-party) Solar Charger to D3 SMART, D5-Evo SMART, and Vx SMART operators.

Improvements

  • D3 SMART Core 1.1.18.0:
  • Added logs to capture the firmware version, making it easier for the technical support team to assist clients
  • Enhanced overall logging functionality
  • Added a recovery mechanism for scenarios where power is lost during an operator format
  • General system and stability improvements
  • D5-Evo SMART Core 1.1.12.0:
  • Added logs to capture Tamper/Theft events
  • Added logs to capture which SMARTGUARDair channel was used, as well as the state of the keypad battery
  • Enhanced overall logging functionality
  • Added a recovery mechanism for scenarios where power is lost during an operator format
  • General system and stability improvements
  • DX Core 3.1.13.0:
  • Added logs to capture Tamper/Theft events
  • Added logs to capture which SMARTGUARDair channel was used, as well as the state of the keypad battery
  • Enhanced overall logging functionality
  • Improved reporting of errors for the Origin sensor during limit setup
  • Added a recovery mechanism for scenarios where power is lost during an operator format
  • General system and stability improvements
  • Vx Core 1.7.11.0:
  • General system and stability improvements
