Release version: 1.5.0.114

Release date: September 04, 2024

Details

Features

  • 1. Allow minimum running speed for 6 m/min on D20
  • 2. Added wiring diagrams for Gatelox.
  • 3. In Trigger Schedules, Holiday Lock is available by default. Previously Holid y Lock would have had to be enabled and an 10 assigned to it for the option to be available in the Schedule Wizard screen.
  • 4. Set PIRAC stop on open distance in mm for VX operators.
  • 5. "Add my profile" feature added to quickly add your number to Operator Admins or MyCentsys Remote under the "Users" section.
  • 6. To reduce the delay of the slave gate in a bi-parting gate installation, we now allow for a 0 second Autoclose on FRX. This only applies to sliding gate operators.

Improvements

  • 1. Resolved the issue where the text to speech would remain active, when disabled by a user.
  • 2. Resolve issue where the LED status would be hidden on SDO4 SMART specifically after an update.
