Release version: 1.5.0.118

Release date: October 17, 2024

Details

Improvements

  • • Auto-populate admin country code:
  • Simplifying the process of adding new administrators, the app now automatically fills in the admin’s country code based on the registered user's location.
  • • Support for Vx SMART power supply options:
  • Added compatibility for low-voltage chargers, expanding installation flexibility for your system.
  • • Positive Close/Open configuration:
  • The app now allows you to configure a minimum push force of 10%, giving you more control over gate movements in Positive Close/Open mode.
  • • Vx SMART:
  • Firmware updated to support low-voltage chargers, ensuring better power management for the system.
  • • Correct AUX output state:
  • Ensures that the default state of the AUX output is set correctly when the controller powers on, preventing potential operational issues.
  • • Gate lock indicator fix:
  • Resolved a flashing issue with the gate lock indicator displayed on the app’s home page.
  • • AC lock overcurrent refinement:
  • Overcurrent protection is now triggered only when 10 consecutive pulses exceed the limit, improving the reliability of the AC lock system.
  • • Optimised logging:
  • The app now logs overcurrent events only on their first occurrence to reduce clutter in system logs.
  • • Double swing gate improvements:
  • Fixed issues on double swing gates where one gate could continue closing while the other retracts in the event of an error, and where the slave gate might try to run before the master when both gates are closed.
  • • Auto-close issue resolved:
  • Corrected a problem where the operator would attempt to auto-close continuously if the system was in a lost state.
  • • Trigger protection:
  • Gate triggers are now blocked when the AC Gate Lock is in Overcurrent Protection Mode, ensuring safer operation.
  • • Remote trigger and gate closing fixes:
  • Addressed issues where remote triggers could be processed twice and where gates could get stuck in the 'closing' state after specific conditions.
  • • Trigger schedule fix:
  • Resolved the issue preventing trigger schedules from being disabled.
  • Device-Specific Improvements:
  • • D3 SMART:
  • General stability improvements have been made to ensure smoother operation.
  • • D5-Evo SMART:
  • General stability improvements have been implemented to enhance system performance.
